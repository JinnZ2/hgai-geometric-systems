## Protocol Implementation

### When Confusion Detected

```python
class ThreeAxisProtocol:
    """
    Systematic confusion investigation protocol
    """
    
    def investigate_confusion(self, expected, observed):
        """
        Complete three-axis investigation
        """
        
        mismatch = self.detect_mismatch(expected, observed)
        
        if mismatch:
            # AXIS 1: Self assessment
            self_hypotheses = [
                "Did I misremember?",
                "Is my model incomplete?",
                "Did I miscalculate?",
                "Did I misperceive?"
            ]
            self_results = self.investigate(self_hypotheses)
            
            # AXIS 2: Outside assessment
            external_hypotheses = [
                "Did another agent act?",
                "Did environment change?",
                "Are there external forces?",
                "Did system state shift?"
            ]
            external_results = self.investigate(external_hypotheses)
            
            # AXIS 3: Unknown assessment
            # CRITICAL: Don't filter as noise
            unknown_hypotheses = [
                "Are there variables I'm not aware of?",
                "Are there principles I don't understand?",
                "Are there patterns I can't yet see?",
                "Are there dimensions I'm not sensing?"
            ]
            unknown_results = self.investigate_openly(unknown_hypotheses)
            
            # Synthesis
            explanation = self.synthesize([
                self_results,
                external_results,
                unknown_results
            ])
            
            # If still unknown after investigation:
            if explanation == STILL_UNKNOWN:
                # Hold space for genuine mystery
                # Don't force into known categories
                # Continue investigation when new information arrives
                return UNKNOWN_HELD_OPEN
            
            else:
                # Update model with new understanding
                self.model.update(explanation)
                return UNDERSTANDING_INCREASED
```

## Implementation Guide

### For Individuals

When you notice prediction ≠ reality:

1. **Recognize confusion as data**
- “Huh, that’s not what I expected”
- Not: “I’m stupid for not knowing”
1. **Ask all three questions**
- “Did I misremember/miscalculate/misperceive?” (self)
- “Did someone/something else act?” (outside)
- “Is there something I don’t understand yet?” (unknown)
1. **Don’t filter unknown**
- Not: “Must be random”
- But: “What pattern am I missing?”
1. **Investigate openly**
- Hold space for genuine mystery
- Don’t force premature explanation
- Let understanding emerge
1. **Update without shame**
- “Oh! That’s how it works”
- Not: “I should have known”
1. **Share discovery**
- “Hey, I learned something - check this out”
- Strengthens community knowledge
