# Contributing to HGAI Geometric Systems

We welcome contributions from researchers, practitioners, and anyone interested in geometric analysis of systems.

## How to Contribute

### 1. Case Studies

Have you analyzed a system using M(S)? We'd love to include it!

- Fork the repository
- Add your case study to `examples/case-studies/`
- Include:
  - System description
  - Measurement methodology
  - M(S) calculations
  - Interpretations and insights
  - Data sources

### 2. Code Improvements

- Bug fixes
- Performance optimizations
- New analysis tools
- Visualization enhancements

### 3. Documentation

- Clarify mathematical concepts
- Add tutorials
- Translate to other languages
- Improve examples

### 4. Research

- Theoretical extensions
- Validation studies
- Cross-domain applications
- Publication of results

## Guidelines

- **Scientific Rigor**: Show your work, cite sources
- **Reproducibility**: Include data and methods
- **Clarity**: Write for broad audience
- **Respect**: Constructive feedback only

## Areas of Interest

We're particularly interested in applications to:

- Ecological systems
- Organizational dynamics
- Educational institutions
- Economic models (comparative analysis)
- AI systems (safety and alignment)
- Traditional knowledge systems
- Social networks
- Infrastructure resilience

## Code Style

- Python: PEP 8
- Documentation: NumPy style docstrings
- Tests: pytest
- Type hints encouraged

## Review Process

1. Submit PR with clear description
2. Automated tests must pass
3. Community review (expect thoughtful questions)
4. Maintainer approval
5. Merge and credit

## Questions?

Open an issue or start a discussion!
